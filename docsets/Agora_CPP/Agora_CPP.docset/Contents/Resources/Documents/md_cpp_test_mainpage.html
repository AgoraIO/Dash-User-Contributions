Agora C++ API Reference for All Platforms
mainpage

声网通过全球部署的虚拟网络，提供可以灵活搭配的 API 组合，为移动端到移动端以及移动端到 Web 端提供质量可靠的实时音视频通信。

  • IRtcEngine 接口类包含应用程序调用的主要方法。
  • RtcEngineParameters 接口类的方法今后将被归入 IRtcEngine 接口类中。目前调用 RtcEngineParameters 类的方法需要先调用 RtcEngineParameters(IRtcEngine& engine) 方法或 RtcEngineParameters(IRtcEngine* engine) 方法创建 RtcEngineParameters 对象。
  • IRtcEngineEventHandler 接口类用于向应用程序发送回调通知。
